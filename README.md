# App reparto horario

Aplicacion web para repartir la carga horaria del Departamento de Matematicas del IES de Teis para el curso 2026/2027.

## Archivo principal

- `App reparto horario.html`: app principal. Es un archivo HTML autosuficiente con estilos y logica incluidos.
- `index.html`: entrada para GitHub Pages. Redirige automaticamente a la app principal.

## Estructura

- `data/`: configuraciones JSON de ejemplo o guardadas.
- `docs/`: documentacion de referencia.
- `backups/`: copias de seguridad de versiones anteriores de la app.

## Uso

Abre `index.html` o `App reparto horario.html` en el navegador. Desde la app se puede exportar e importar la configuracion en formato JSON.

## Publicacion

El repositorio esta preparado para publicarse con GitHub Pages desde la rama principal.
